# Sandbox Theme Recognition 沙盘主题识别

## 代码说明

用三种不同的网络来进行主题识别的实验
- 自定义网络、resnet34、clip对应的脚本分别为main.py、main_resnet34.py、main_clip.py

- runs、runs_clip、runs_resnet34分别为自定义网络、clip、resnet34对应的tensorboard的可视化的结果

- checkpint文件夹里对应的为训练完的模型参数
